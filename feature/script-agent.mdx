## 脚本策划

**1. 点开智能策划，上传产品图片，给出产品描述，输入需求**

<Frame>
  ![Script Agent 1](/images/script-agent-1.webp)
</Frame>

**2. 对脚本信息进行个性化的修改，确认产品与脚本的基础信息**

<Frame>
  ![Script Agent 2](/images/script-agent-2.webp)
</Frame>

**3. 点击“生成视频”按钮直接跳转到视频生成界面**

<Frame>
  ![Script Agent 3](/images/script-agent-3.webp)
</Frame>

**4. 上传产品图，@产品图进行引用，按需求修改提示词，生成视频**

    </Frame>
  </Column>

  <Column>

  </Column>
  <Column>

      ![Script Agent 10 Upload Asset](/images/script-agent-10-upload-asset.webp)
    </Frame>
  </Column>
</Columns>

**2. 脚本拆解完成后，点击“一键复刻”**

<Frame>
  ![Script Agent 11 Click](/images/script-agent-11-click.webp)
</Frame>

**3. 上传产品图，填写要求与基础信息**

<Frame>
  ![Script Agent 9 Upload Ref](/images/script-agent-9-upload-ref.webp)
</Frame>

**4. 确认产品信息和脚本信息，补充信息，修改要求**

<Frame>
  ![Script Agent Comfirm](/images/script-agent-comfirm.webp)
</Frame>

<Frame>
  ![Script Agent Gen Video](/images/script-agent-gen-video.webp)
</Frame>

**6. 添加产品图与模特图，@【图片】进行添加，生成视频，并点击“一键复刻**
